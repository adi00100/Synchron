# Set Up
- Create .env with following environment variables
```
ENGINE=django.db.backends.postgresql
NAME=...
USER=...
PASSWORD=...
HOST=...
PORT=...
```
- Create a database in postgres and create schema by running the DDL queries in  `database-queries.ipynb`
